Editorfenster


Dies ist das Fenster zum erstellen eines neuen Spiels.


In der Menüleiste finden sich folgende Punkte:


Durch Klick auf den Button "Feld erstellen" öffnet sich ein Untermenü, in dem Sie zunächst die Größe des Spielfeldes einstellen können.
Über den Button "Start" wird ein blankes Spielfeld in der zuvor gewählten Größe erstellt.
Zur Bedienung des Editor sehen Sie bitte unter "Bedienung -> Editor" nach.



Über den Button "Zurücksetzen" wird das gesamte bisher erstellte Spielfeld verworfen.
Sie befinden sich dann im leeren Editorfenster.



Die Funktion "Regeln prüfen" prüft das erstellte Spielfeld darauf, dass keine Zelle unbesetzt ist und das Spiel lösbar ist.



Über den Button "Feld speichern" gelangen Sie in ein Menü, in dem Sie das erstellte Spiel abspeichern können.



Über den Button "Feld laden" gelangen Sie in ein Menü, in dem Sie aus bereits erstellten Spielen eines auswählen können, um es zu bearbeiten.



Über den Button gelangen Sie zurück zum Hauptmenü.



Der Button beendet das Spiel.





Zum Starten eines Spiels wählen Sie bitte zunächst "Spiel laden" und wählen ein Spiel aus.
Haben Sie ein Spiel geladen, finden Sie im Fenster das Spielfeld und eine Liste aller getätigten Züge (rechte Seite).
