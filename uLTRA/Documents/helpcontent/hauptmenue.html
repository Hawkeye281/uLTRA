Hauptmenü


Dies ist der Startbildschirm des Spiels.
Von hier aus haben Sie Zugriff auf alle Funktionalitäten.



Über den Button "Neues Spiel" gelangen Sie in ein zufällig generiertes Spielfeld. Dieses Spiel können Sie jederzeit speichern (siehe Spielfenster -> Spiel speichern).



Über den Button "Editor starten" gelangen Sie in dem Sie selber Spiele erstellen können (siehe Editor).



Der Button "Spiel beenden" beendet das Spiel.
