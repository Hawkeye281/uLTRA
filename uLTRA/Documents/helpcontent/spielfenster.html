Spielfenster


Dies ist das Fenster für ein Spiel.


In der Menüleiste finden sich folgende Punkte:

Über den Button "Spiel laden" gelangen Sie in ein Menü, in dem Sie selbst erstellte Spiele (siehe Editorfenster) oder abgespeicherte Spiele öffnen bzw. fortsetzen können.



Über den Button "Spiel speichern" gelangen Sie in ein Menü, in dem Sie den aktuellen Spielstand abspeichern können.



Betätigen Sie den Button "Spiel lösen", so wird das aktuelle Spiel für Sie gelöst.



Über den Button gelangen Sie zurück zum Hauptmenü.



Der Button beendet das Spiel.



Zum Starten eines Spiels wählen Sie bitte zunächst "Spiel laden" und wählen ein Spiel aus.
Haben Sie ein Spiel geladen, finden Sie im Fenster das Spielfeld und eine Liste aller getätigten Züge (rechte Seite).
