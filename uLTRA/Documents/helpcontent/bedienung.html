Spielfeld


Zum Erstellen eines Lichtstrahls im Spielfeld ziehen Sie bei gedrückter LINKER Maustaste vom Startpunkt in die gewünschte Richtung. Dabei können Sie eine beliebige Länge ziehen (max. des Startpunktes beachten!).

Sollten Sie beim Ziehen eines Lichtstrahls einen Fehler gemacht haben, so können Sie - wieder vom Startpunkt aus - mit gedrückter und gehaltener RECHTER Maustaste alle Strahlen in einer Richtung durch ziehen in diese löschen.



Editor


Zum Erstellen eines eigenen Spielfeldes wählen Sie im Editor zunächst "Feld erstellen".
Es öffnet sich ein Untermenü, in dem Sie die Größe des Spielfeldes festlegen können (3x3 - 20x20).
Nach getätigter Eingabe drücken Sie "Start". Es wird ein leeres Spielfeld in der gewählten Größe angezeigt.

Durch doppeltes Klicken mit der linken Maustaste in ein leeres Kästchen generieren Sie einen Startpunkt.
Dieser Startpunkt hat standardmäßig den Wert "0". Sie können im Anschluß von diesem Startpunkt aus Strahlen in beliebige Richtungen ziehen (klicken und halten während des Ziehens in eine Richtung).
Haben Sie einen Lichtstrahl abgeschlossen, wird der Wert innerhalb des Startpunktes automatisch an die Gesamtzahl der von ihm abgehenden Strahlen angepasst.

Möchten Sie eine Eingabe korrigieren, haben Sie zwei Möglichkeiten:
  • Löschen eines Startpunktes und aller abgehenden Strahlen:
    Für diese Löschung klicken Sie mit RECHTER Maustaste auf den zu löschenden Startpunkt und ziehen mit gedrückter Maustaste in eine beliebige Richtung.
  • Verändern eines einzelnen Lichtstrahls:
    Um lediglich einen Strahl zu ändern klicken Sie (vom Ende des Strahls an) mit rechter Maustaste auf das zu löschende Teilstück des Strahls.
